Description
This project uses Django as its web framework and Open-Meteo
as the weather API.

Instructions
1.  Install Django
    python -m pip install Django

2.  cd to the \WeatherApp\weather_project directory

3.  Use the following command:
    python manage.py runserver

* Also make sure that Pandas is installed.
